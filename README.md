# Dexetera Documentation

This repository contains the official documentation website for **[Dexetera](https://dexetera.org)** — a gas-free decentralized exchange (DEX) built on Hyperliquid that lets you trade any measurable market.

> **Live links:** [dexetera.org](https://dexetera.org) (homepage) · [dexetera.xyz](https://dexetera.xyz) (trading app)

---

## What is Dexetera?

Dexetera is a fully decentralized, non-custodial trading platform where users can:

- Trade **commodities, stocks, indexes**, and other real-world markets
- **Create custom futures contracts** (1-year expiration) on any verifiable metric
- Trade **gas-free** with a single click using **USDC**
- Maintain full self-custody — no company can access your funds

Built on top of **Hyperliquid**, Dexetera combines a fully on-chain order book with a seamless, one-click trading experience.

## Core Principles

### Unrestricted Markets
Trade what you believe in. From global commodities and stock indexes to hyper-local metrics — if it's measurable and verifiable, it can be a market on Dexetera.

### True Decentralization
A fully on-chain order book with no intermediary. Your capital stays in your wallet; Dexetera never takes custody of your funds.

### Complete Transparency
Every trade, settlement, and operation is recorded on-chain and publicly verifiable. Immutable history, open and traceable.

## About This Docs Site

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

### Installation

```bash
npm install
```

### Local Development

```bash
npm start
```

Most changes are reflected live without having to restart the server.

### Build

```bash
npm run build
```

Generates static content into the `build` directory, ready to be served by any static hosting service.

### Deployment

The docs site runs on a remote server at `/home/ddy/dexetera-docs`, exposed on port **6902**.

Deploy using the provided script:

```bash
./deploy.sh
```

Or serve manually:

```bash
npm run serve
```
